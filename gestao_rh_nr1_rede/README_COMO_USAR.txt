GESTÃO RH CONSULTORIA - PLATAFORMA NR-1 EM REDE LOCAL

COMO ABRIR:
1. Extraia esta pasta em um computador que ficará ligado.
2. Clique em ABRIR_SISTEMA_REDE.bat.
3. Abra no navegador o endereço mostrado, exemplo: http://192.168.1.50:3000
4. Use esse IP para celulares e outros computadores da mesma rede.

FLUXO DO SISTEMA:
- Gestão RH acessa o painel administrador.
- Em "Perguntas", o administrador pode editar o questionário.
- Ao salvar perguntas, o sistema cria uma NOVA VERSÃO.
- Links e empresas já criadas continuam com a versão antiga das perguntas.
- Novas empresas criadas depois disso usam a versão nova.
- O link do funcionário mostra somente o formulário.
- O link do patrão/gestor mostra somente o dashboard da empresa dele.

DADOS:
- As informações ficam salvas no arquivo data.json.
- Não apague esse arquivo se quiser manter empresas, respostas e versões dos questionários.

OBSERVAÇÃO:
Esta é uma versão local/rede. Para acesso pela internet, uso por várias empresas fora da mesma rede e login com senha, o próximo passo é colocar em servidor com banco de dados.
